Terms of Service

Last updated: May 3, 2026

1. About the Application

Nour Quran ("the App") is an automated content publishing application that generates and publishes Quran recitation videos with French translations to social media platforms including YouTube and TikTok. The App is operated by an individual creator for religious and educational purposes.

2. Use of TikTok API

The App uses the TikTok Content Posting API (video.publish scope) exclusively to publish videos directly to a single authorized TikTok creator account. The App does not interact with any third-party users' TikTok accounts.

3. Content

All video content published by the App consists of Quran recitations by Mishary Rashid Al-Afasy with synchronized French translations. The content is intended for religious education and spiritual enrichment.

4. Intellectual Property

Audio recitations are sourced from publicly available Islamic audio archives. All content is used for non-commercial religious and educational purposes.

Privacy Policy

Last updated: May 3, 2026

1. Introduction

This Privacy Policy describes how Nour Quran handles information in connection with your use of our services.

2. Information We Do Not Collect

Nour Quran does not collect, store, process, or share any personal data. Specifically:

  • No user accounts or registrations are required
  • No personal information is collected
  • No cookies or tracking technologies are used
  • No analytics data is gathered from viewers
  • No third-party data sharing occurs

3. TikTok API Usage

The App uses the TikTok Content Posting API (video.publish scope) to automatically publish video content to a single authorized creator account. The App:

  • Only accesses the creator's own TikTok account
  • Does not access, store, or process any other user's TikTok data
  • Does not collect any data through the TikTok API
  • Uses only the video.publish scope to upload videos directly

4. Data Storage

The App stores only the video files generated for publication locally on the operator's server. No personal data is stored at any point.

5. Third-Party Services

The App integrates with the following third-party APIs:

  • TikTok Content Posting API — video.publish scope for direct video publication
  • YouTube Data API — for video publication
  • OpenAI API — for metadata generation
  • Pexels API — for background images

6. Children's Privacy

The App does not collect any data from children or any other users. Our content is suitable for all ages.
