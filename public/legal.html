This Privacy Policy describes how the Nour application handles information in connection with your use of our services.

The Nour application does not collect, store, process, or share any personal data from users. Specifically:

  • No personal information is collected
  • No cookies or tracking technologies are used

3. TikTok API Usage

The App uses the TikTok Content Posting API to automatically publish video content to a single authorized creator account. The App:

  • Only accesses the creator's own TikTok account

The App stores only the video files generated for publication (Quran recitation videos) locally on the operator's server. No personal data is stored at any point.

Each of these services has its own privacy policy. No user data is shared with these services.
